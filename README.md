# Nice Views

Interactive map of scenic viewpoints, hidden gems, waterfalls, beaches, and mountain vistas — all within 30 miles of Chelmsford, MA. 

**[View the live map →](https://jdinino.github.io/NiceViews/)**

## What's Here

21 curated spots organized by route:

- **Local Loop** — 7 spots you can hit in an afternoon
- **Ponds, Falls & Mountains** — Trap Falls, Wachusett Mountain, Walden Pond, and more
- **Boston & Beaches** — Singing Beach, Winter Island, Forest River Park
- **Directional routes** — Going North, South, East, or West from Chelmsford

Each spot includes GPS coordinates, What3Words address, Google Maps link, Street View link, and a Wikipedia reference photo.

## Features

- Leaflet map with category-specific icons (mountain, waterfall, beach, pond, river, lookout, historic, park)
- Route filtering with automatic map zoom
- Route Planner tab with ordered stop sequences by direction
- Sortable spot table with clickable GPS and What3Words links

## Perfect For

- October foliage drives
- Weekend day trips
- Photography outings
- Any time you need a view that'll make you stop and breathe
