# Nice Views

Interactive map of scenic viewpoints, hidden gems, waterfalls, beaches, mountain vistas, motorcycle routes, and public restrooms across Massachusetts and New Hampshire — anchored on Chelmsford, MA.

**[View the live map →](https://jdinino.github.io/NiceViews/)**

## What's on the Map

| Layer | Count | Source |
| --- | --- | --- |
| Curated scenic spots | 35 | Hand-picked within ~30 mi of Chelmsford |
| NER scenic views, mountains, waterfalls, historic + park sites | ~115 | [New England Riders B-O-N-E](https://www.newenglandriders.org/b-o-n-e/) |
| Motorcycle-friendly road segments (with OSM polylines where matched) | 100 | NER B-O-N-E + [OpenStreetMap](https://www.openstreetmap.org) via Overpass |
| Public restrooms | 810 | [City of Boston Public Amenities Map](https://www.google.com/maps/d/viewer?mid=1yciPnqgJqtqKFcZI4NAvRm6ey9A_wT0) + [bathroomaccess.com](https://bathroomaccess.com) |

## Filters

**Radio (one at a time):**
- **All Spots** — every scenic spot on the map
- **Within 30 mi** — anything within 30 miles of Chelmsford (the original curated scope)
- **Local Loop** — Chelmsford-area afternoon trip
- **Ponds, Falls & Mountains** — Wachusett, Walden Pond, Trap Falls, and more
- **Boston & Beaches** — Singing Beach, Winter Island, Forest River Park, and Boston-area picks
- **Going North / South / East / West** — directional tours from Chelmsford

**Toggle layers (overlay independently on top of any radio filter):**
- 🚻 **Restrooms** — 810 public restrooms
- 🏍️ **Riders' Roads** — motorcycle routes with road geometry highlighted

## Spot Details

Each marker opens a sheet with:
- GPS coordinates and a Google Maps link
- Street View link
- What3Words address (where available)
- Wikipedia reference photo (where matched)

## Features

- Leaflet map with category-specific icons (mountain, waterfall, beach, lookout, historic, park, canal, road, restroom)
- Live GPS location: auto-locate on load, pulsing blue dot, recenter button
- Fullscreen map toggle (top-right) for mobile real-estate
- Route Planner tab with ordered stop sequences by direction
- Sortable spot table with clickable GPS and What3Words links
- Mobile-first responsive layout with horizontal pill-bar filters
